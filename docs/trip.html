【輕鬆假期】北台灣知性２日遊

參考售價 : 費用按包車人數及車輛種類計算。

行程編號：98GO02001

行程特色

  • 特別安排國際級故宮博物館半天文化巡禮
  • 結合北台灣最熱門的三大景點：九份、十分、淡水老街
  • 專業有執照司導全程介紹
DAY 1
第一天
台北 +++ 中烈祠 → 故宮博物院 → 淡水老街 → 士林夜市 → 飯店休息
早餐後前往忠烈祠參觀著名的衛兵交接。隨後前往故宮博物館參觀，一賭翠玉白菜等千年的中國古物。
午後續前往愛情聖地漁人碼頭和淡水河出口的淡水老街遊覽。傍晚前往台灣最大的觀光夜市『士林夜市』。
餐食：早-敬請自理　 午-風味午餐　 晚-敬請自理 住宿：台北地區優質飯店
DAY 2
第二天
台北 +++ 十分老街 → 九份 → 野柳 → 台北西門町→飯店(結束旅程)
早餐後前往十分老街體驗放天燈的樂趣。隨後前往九份老街，欣賞山城美景和美味小吃。
午後續往黃金瀑布和著名的石灰岩地型『野柳』遊覽並與女王頭合影。
傍晚可選擇基隆夜市小吃或返回台北西門町逛街，之後返回酒店結束一天行程。
